## 1. 环境要求

| 组件 | 推荐版本 | 说明 |
|---|---|---|
| JDK | 17+ | 项目 Java 版本为 17 |
| Maven | 3.8+ | 本机默认 Maven 3.6.2 过低，建议使用 3.8.8 |
| MySQL | 8.x | 业务数据库 |
| Redis | 6.x+ | 大纲临时状态缓存 |
| Nacos | 2.x | 配置中心和服务发现 |
| 操作系统 | Windows / Linux | 本文命令以 Windows PowerShell 为例 |

检查命令：

```powershell
java -version
mvn -version
```

如果默认 `mvn` 版本低于 3.6.3，可直接使用本机 Maven 3.8.8：

```powershell
& 'C:\apache-maven-3.8.8\bin\mvn.cmd' -version
```

## 3. 数据库初始化

### 3.1 创建数据库

登录 MySQL 后创建数据库：

```sql
CREATE DATABASE IF NOT EXISTS power_report
  DEFAULT CHARACTER SET utf8mb4
  COLLATE utf8mb4_unicode_ci;
```

### 3.2 执行建表脚本

在仓库根目录下执行：

```sql
SOURCE ./report-generation/database/schema_mysql.sql;
```

核心表：

| 表名 | 说明 |
|---|---|
| `reports` | 报告主表 |
| `report_outline_nodes` | 大纲节点表 |
| `report_sections` | 章节正文表 |
| `report_files` | 导出文件记录表 |
| `report_templates` | 模板表 |

## 4. Redis 准备

Redis 用于保存 AI 生成大纲的临时状态。

本地默认配置示例：

```properties
spring.data.redis.host=127.0.0.1
spring.data.redis.port=6379
spring.data.redis.database=0
```

如果 Redis 配置密码，需要补充：

```properties
spring.data.redis.password=your-redis-password
```

## 5. Nacos 配置

`report-outline-docx/src/main/resources/application.properties` 中只保留 Nacos 引导配置，业务配置建议放在 Nacos。

Nacos 新建配置：

```text
Data ID: report-outline-docx
Group: DEFAULT_GROUP
Format: Properties
```

配置内容可参考：

```text
report-generation/report-outline-docx/src/nacos-config-example.properties
```

推荐配置：

```properties
server.port=8080

spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
spring.datasource.url=jdbc:mysql://127.0.0.1:3306/power_report?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai&useSSL=false&allowPublicKeyRetrieval=true
spring.datasource.username=your-mysql-username
spring.datasource.password=your-mysql-password

spring.data.redis.host=127.0.0.1
spring.data.redis.port=6379
spring.data.redis.database=0

mybatis-plus.configuration.map-underscore-to-camel-case=true
mybatis-plus.global-config.db-config.id-type=input

app.report.export-dir=./storage/reports
app.ai.outline-url=http://127.0.0.1:9000/api/ai/outline/generate
app.ai.timeout-seconds=60
app.ai.outline-temp-ttl-seconds=1800
app.ai.fallback-enabled=true

management.health.redis.enabled=false
management.health.db.enabled=false
```

配置项说明：

| 配置项 | 说明 |
|---|---|
| `server.port` | 后端服务端口 |
| `spring.datasource.*` | MySQL 连接配置 |
| `spring.data.redis.*` | Redis 连接配置 |
| `app.report.export-dir` | DOCX 本地导出目录 |
| `app.ai.outline-url` | AI 大纲服务地址 |
| `app.ai.fallback-enabled` | AI 不可用时是否使用本地模板兜底 |

联调建议：

- 本地演示可设置 `app.ai.fallback-enabled=true`。
- 验证 AI 真实调用时设置 `app.ai.fallback-enabled=false`。

## 7. 接口联调顺序

建议按以下顺序联调：

| 顺序 | 接口 | 目的 |
|---:|---|---|
| 1 | `GET /api/health` | 确认服务启动 |
| 2 | `POST /api/reports/outline` | 验证本地固定大纲 |
| 3 | `POST /api/reports/outline/generate` | 验证 AI 生成或本地兜底 |
| 4 | `POST /api/reports/outline/confirm` | 保存报告和大纲 |
| 5 | `POST /api/reports/{reportId}/export/docx` | 生成 DOCX |
| 6 | `GET /api/reports/files/{fileId}/download` | 下载 DOCX |

## 9. 常见问题

### 9.1 Maven 版本过低

现象：

'''text
maven-compiler-plugin requires Maven version 3.6.3
```

处理：

```powershell
& 'C:\apache-maven-3.8.8\bin\mvn.cmd' -version
```

使用 Maven 3.8.8 或更高版本重新执行。

### 9.2 Nacos 未启动

现象：

- 服务无法读取业务配置。
- 数据库、Redis、端口配置不生效。

处理：

- 启动 Nacos。
- 确认 `Data ID=report-outline-docx`。
- 确认 `Group=DEFAULT_GROUP`。

### 9.3 数据库连接失败

检查：

- MySQL 是否启动。
- 数据库 `power_report` 是否存在。
- 用户名、密码是否正确。
- 是否执行 `schema_mysql.sql`。

### 9.4 Redis 连接失败

检查：

- Redis 是否启动。
- 端口是否为 `6379`。
- 如有密码，是否配置 `spring.data.redis.password`。

### 9.5 DOCX 无法下载

检查：

- 是否已先执行导出接口。
- `report_files` 表中是否存在文件记录。
- `filePath` 指向的本地文件是否存在。
- `app.report.export-dir` 是否有写权限。
